## The thesis

I close the gap between idea and artifact, then stay accountable for whether anyone uses it. The translation layer — idea → ticket → sprint → something that isn't what was named in the room — is the cost I build to eliminate. Everyone has the same tools now; the tool was never the differentiator. The person using it is.

## Stack

Static HTML/CSS/JS. No build step. Inter / JetBrains Mono / Caveat. Deployed via GitHub Pages.
